
# Gesti-n_Alquiler_Autos_ReactVite

## Descripción
Aplicación web para la gestión de alquiler de autos. Permite registro, login y funcionalidades de alquiler de vehículos.  

## Tecnologías
**Frontend:** React, Vite, PrimeReact, React Router DOM  
**Backend:** Node.js, Express, Sequelize, JWT, bcryptjs  
**Base de datos:** MySQL  

## Instalación rápida

1. Clonar el repositorio:  
```bash
git clone https://github.com/RodoGitHub/Gesti-n_Alquiler_Autos_ReactVite.git
cd Gesti-n_Alquiler_Autos_ReactVite
```

2. Configurar variables de entorno:  
```bash
cp .env.example .env
```
Editar `.env` con tus datos:
```
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=alquiler_autos
JWT_SECRET=TuClaveSecreta
```

3. Instalar dependencias:  
```bash
npm install
```

4. Levantar la aplicación:  
```bash
npm run dev
```
